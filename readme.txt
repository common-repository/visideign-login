---- Installation ----

1. Upload the `visideign-login` directory (including all files within) to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. Go to `Appearance` -> `Widgets` and drag the `visideign-login` widget to your sidebar
